    # 해결

    Xcode Full Install 하면 해결된다고 하지만 Full Install 하고 싶지 않았다.

    Xcode tools를 재 설치하면 된다는 글 (opens new window)을 확인해서 시도해봤다.

    $ sudo rm -rf $(xcode-select -print-path)
$ xcode-select --install
    1
    2

    위 명령어를 실행후 다시 npm install하니 잘된다. 끗.
