# 1. Hyperledger Fabric Docker Images

# 1.1 하이페레저 패브릭 관련 도커 이미지

  • hyperledger/fabric-ca (opens new window)

    CA 서버 이미지

  • hyperledger/fabric-zookeeper (opens new window)

    멀티 Orderer를 지원 / MQ 환경을 구축하는데 필요한 Zookeeper

  • hyperledger/fabric-kafka (opens new window)

    멀티 Orderer를 지원 / MQ 환경을 구축하는데 필요한 Kafka

  • hyperledger/fabric-nodeenv (opens new window)

    Node.js 용 Chaincode를 구동하기 위한 이미지

  • hyperledger/fabric-javaenv (opens new window)

    Java 용 Chaincode를 구동하기 위한 이미지

  • hyperledger/fabric-ccenv (opens new window)

    Go 용 Chaincode를 구동하기 위한 이미지

  • hyperledger/fabric-tools (opens new window)

    Blockchain 네트워크를 관리하고 스마트컨트랜트 관리 설치하는 기능을 제공하는 이미지

  • hyperledger/fabric-orderer (opens new window)

    Hyperledger Fabric Orderer(블록 생성 담당) 이미지

  • hyperledger/fabric-peer (opens new window)

    Hyperledger Fabric Peer(트랜잭션을 발생시키고 Chaincode를 실행하며 원장을 가지는 녀석) 이미지

  • hyperledger/fabric-baseos (opens new window)

    Hyperledger Fabric 기본 이미지로 Chaincode 구동할때 사용
